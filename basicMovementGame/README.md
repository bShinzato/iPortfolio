# basicMovementGame

A small 2D browser game built with React, TypeScript, and Vite.

The goal of this project is to create a Final Fantasy-inspired mini adventure with:
- WASD movement
- a canvas-based game world
- React UI for text and overlays
- simple game logic that grows over time

## How to run

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

If the browser does not open automatically, use the local URL shown in the terminal.

## Controls

- `WASD` - move the player
- `keydown` / `keyup` input is tracked by the game loop
- More controls will be added as the game grows

## Tech Stack

- React
- TypeScript
- Vite
- HTML Canvas

## Project Status

This project is still in early development. The current focus is building the movement system, canvas rendering, and simple game structure step by step.
